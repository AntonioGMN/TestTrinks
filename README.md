# TestTrinkes

## How to run

1. Clone this repository

2. Install dependencies

```bash
npm install
```
3. Run the prisma

```bash
npx prisma migrate dev
```

4. Run the seed of the database

```bash
npm run seed
```

5. Or run the app in development mode

```bash
npm run dev
```

G. Run the tests

```bash
npm run test
```
